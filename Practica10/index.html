Mi página con bootstrap

Bootstrap es un popular marco de desarrollo web front-end de código abierto que permite a los desarrolladores crear aplicaciones web receptivas y móviles de forma rápida y sencilla. Bootstrap incluye componentes y utilidades HTML, CSS y JavaScript preconstruidos, como barras de navegación, formularios, botones, tipografía y sistemas de cuadrícula, que se pueden personalizar y ampliar fácilmente.
